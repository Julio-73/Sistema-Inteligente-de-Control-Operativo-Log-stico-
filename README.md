# 🚚 Sistema Inteligente de Control Operativo Logístico

![Python](https://img.shields.io/badge/Python-3.11-blue?logo=python)
![Streamlit](https://img.shields.io/badge/Streamlit-1.54-red?logo=streamlit)
![Pandas](https://img.shields.io/badge/Pandas-2.3-green?logo=pandas)
![Estado](https://img.shields.io/badge/Estado-Producción-brightgreen)

> Dashboard empresarial para el análisis y control de operaciones logísticas en tiempo real. 
> Transforma datos operativos en decisiones estratégicas, reduciendo 15+ horas semanales de trabajo manual.

🌐 **Demo en vivo:** [sistema-logistico.streamlit.app](https://sistema-logistico.streamlit.app/)


## ✨ Características

- 📊 KPIs ejecutivos en tiempo real
- ⚠️ Alertas automáticas de negocio
- 🔍 Filtros por Ruta, Conductor y Estado
- 📈 Gráficos interactivos con Plotly
- 📥 Exportación a Excel
- 📄 Reporte Ejecutivo PDF corporativo
- ☁️ Desplegado en la nube
